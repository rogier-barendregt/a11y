## ARIA

ARIA (Accessible Rich Internet Applications) is een technische specificatie ontwikkeld door W3C om webinhoud en applicaties toegankelijker te maken voor mensen met een beperking — vooral voor gebruikers van hulpmiddelen zoals schermlezers.

Met ARIA kun je extra toegankelijkheidsinformatie toevoegen aan HTML-elementen via speciale attributen. Zo kunnen dynamische of interactieve elementen die van nature niet toegankelijk zijn met standaard HTML beter beschreven worden en geeft het semantische betekenis en statusinformatie door aan hulpmiddelen.
